# Starv Performance — Тюнинг-ателье

АИС для автоматизации работы тюнинг-ателье: каталог компонентов, премодерация заказов и запись на сервис.

## Стек технологий
- **Frontend:** React.js
- **Backend:** Laravel (PHP)
- **Database:** MySQL

## Запуск проекта
1. **Backend:**
   - `cd backend`
   - `composer install`
   - `php artisan serve`
2. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm run dev`

## Макет интерфейса (Figma)

### Описание макета
Макет реализован в темной теме для соответствия премиальному бренду. 
**Реализованные экраны:**
1. Главная (Landing)
2. Каталог запчастей (с фильтрами)
3. Каталог услуг
4. Корзина (с полем ввода VIN для проверки совместимости)
5. Админ-панель: Список заявок и модерация
5. Админ-панель: Управление товарами
5. Админ-панель: Управление услугами. 

### Пользовательские сценарии
1. **Подбор тюнинга:** Клиент заходит в каталог -> выбирает детали -> вводит VIN в корзине -> отправляет на проверку.
2. **Модерация:** Админ заходит в панель -> видит новую заявку -> проверяет совместимость -> меняет статус на "Проверено".
