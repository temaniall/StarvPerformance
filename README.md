# Starv Performance — Тюнинг-ателье

АИС для автоматизации работы тюнинг-ателье: каталог компонентов, премодерация заказов и запись на сервис.

## Стек технологий
- **Frontend:** React.js
- **Backend:** Laravel (PHP)
- **Database:** MySQL

## Запуск проекта
1. **Backend:**
   - `cd backend`
   - `composer install`
   - `php artisan serve`
2. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
